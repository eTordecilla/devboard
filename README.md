# DevBoard

Plataforma de gestión de proyectos técnicos construida con el stack MERN
(MySQL, Express, React, Node.js).

## Stack tecnológico

- **Backend:** Node.js 22 LTS + Express
- **Frontend:** React + Vite
- **Base de datos:** MySQL
- **Autenticación:** JWT + bcrypt

## Requisitos previos

- Node.js 22 LTS (via NVM)
- MySQL 8+
- Git

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/devboard.git
cd devboard

# Backend
cd backend
cp .env.example .env    # Completá las variables
npm install
npm run dev

# Frontend (en otra terminal)
cd frontend
npm install
npm run dev
```
